NeetCode

This project is a platform similar to LeetCode, built using Express, Node.js, React, MongoDB, Daisy UI, and Redis. The platform includes user authentication features such as login and signup.

Features

User Authentication: Signup, Login, and Authentication using JWT tokens.

Problem Solving: Users can view, attempt, and submit solutions for coding problems.

Leaderboard: Users can view rankings based on problem-solving progress.

Session Management: Redis is used to manage user sessions and enhance performance.

Tech Stack

Frontend: React.js, Daisy UI

Backend: Node.js, Express

Database: MongoDB (for storing user data, problems, and submissions)

Session Management: Redis

Authentication: JWT (JSON Web Tokens)
